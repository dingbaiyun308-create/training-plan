iPhone 使用方式

当前版本：
第2阶段：4周“3＋1”力量加速、背部强化与首个引体计划。

上传到 GitHub Pages：
把本文件夹里的这些内容上传并覆盖到 GitHub Pages 仓库根目录：
index.html
manifest.webmanifest
sw.js
icons 文件夹

手机端安装：
1. 用 iPhone Safari 打开 GitHub Pages 网址。
2. 点分享按钮。
3. 选择“添加到主屏幕”。
4. 后续从主屏幕“训练计划器”打开。

数据保存：
打勾、实际重量、恢复记录、引体进度会保存在当前 iPhone 的 Safari 本地存储。
更新版本或换手机前，请先在网页中点“导出备份”。
更新后打开新版网页，再点“导入”恢复记录。

如果手机仍显示旧计划：
先刷新 Safari 页面。
如果主屏幕图标仍旧，请删除旧主屏幕图标后重新添加。
